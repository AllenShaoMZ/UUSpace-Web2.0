# 卫星综测 Web 端对照开发表

本文用于把桌面版 `D:\UUSpace1.0.0\SateliteController` 的程序设计、UI 绑定和 Web 端实现一一对齐。Web 端不复刻旧软件界面，只保留需要的业务能力：连接、协议配置、遥测表格、遥测曲线、状态、指令。

## 总体对应

| 软件版模块/文件 | 软件版职责 | Web 对应 | 当前状态 | 继续开发要点 |
| --- | --- | --- | --- | --- |
| `SateliteController\MainForm.cs` | 主窗体、菜单、dock、UDP 数据调度、表格/曲线刷新 | `index.html` + `app.js` Mission Workspace | 已建立 | 保持顶部任务栏 + 左摘要 + 中央工作区 + 右检查器布局 |
| `MainForm.Designer.cs` | 顶部菜单、模式按钮、工具按钮 | `.mission-bar`、`.workspace-tabs` | 已建立 | 不照搬 WinForms 控件，只映射功能入口 |
| `Protocol\DataGenerateUDP.cs` | UDP 监听、端口/规则匹配、数据派发 | `tools/udp_web_server.py` | 已实现 7101-7108 监听 | 后续可补完整包头、包长、校验策略 |

## Web 功能完成情况

| 功能 | Web 文件 | 操作方式 | 状态 |
| --- | --- | --- | --- |
| 局域网打开网页 | `tools/udp_web_server.py` | `--host 0.0.0.0 --http-port 8080` | 已支持 |
| UDP 数据验证 | `/api/udp/status`、`/api/udp/events` | 连接管理页查看最近包、端口包数、HEX | 已支持 |
| 读取全量遥测大表 | `/api/telemetry/definitions` | 前端启动后自动拉取 | 已支持 |
| 每个端口对应一个 Sheet 表格 | `state.activeSheet` + Sheet 标签 | 遥测表格页切换 Sheet0-Sheet7 | 已支持 |
| 查看表格刷新 | `sheetStats`、`sheetLiveValues` | 表格顶部显示包数、更新时间、刷新状态 | 已支持 |
| 添加表格 | `tableViews` | 遥测表格页点击“添加表格” | 已支持基础版 |
| 添加曲线 | `channels`、`curveBuffers` | 表格选中参数后点“添加曲线”，或曲线页勾选 | 已支持基础版 |
| 状态总览 | `renderStatus()` | 显示链路、告警、刷新状态 | 已支持 |
| 协议配置 | `renderProtocol()` | 查看 S0-S7 包头、包长、端口、Sheet | 已支持展示 |
| 指令控制 | `renderCommandCenter()` | 指令卡片、发送确认 | 已支持静态版 |

## 当前不做

| 项目 | 原因 |
| --- | --- |
| 回放 | 用户明确不需要 |
| 旧软件 WinForms 布局复刻 | 用户不喜欢旧界面，Web 按 md 文档 Mission Workspace 设计 |
| 完整指令链/历史回放 | 当前需求只强调“需要指令”，先保留指令控制入口 |

## 下一阶段建议

1. 将协议配置从静态数组改为可保存的 JSON 配置。
2. 把指令发送接入后端接口，增加二次确认和发送结果状态。
3. 表格列配置、添加表格、曲线通道保存到浏览器 localStorage 或后端配置。
4. 曲线绘制可替换为 ECharts/uPlot/SciChart JS，支持缩放、暂停、导出。
