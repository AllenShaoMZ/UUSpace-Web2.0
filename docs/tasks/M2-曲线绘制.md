# M2 — 曲线绘制

| 属性 | 内容 |
|------|------|
| **阶段** | P1（C-01/02）+ P2（C-03/04/05）+ P4（C-06） |
| **依赖** | MC、M0；与 M1 勾选通道联动 |
| **主要文件** | `app.js`（`renderCurve`）、`index.html`（引入 ECharts）、`styles.css` |

## 目标

用 **ECharts 5** 替换 Canvas 主路径；保持现有多面板网格与 Mission Workspace 视觉；工具栏控制运行/暂停（无 F5/F6）。

### UI 体验（[UI-交互增强.md](./UI-交互增强.md)）

- [x] **UI-03** 曲线 Tab 箭头 + 滚轮横向浏览（无横向滚动条）
