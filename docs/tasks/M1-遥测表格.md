# M1 — 遥测表格

| 属性 | 内容 |
|------|------|
| **阶段** | P1（T-01/02/04）+ P2（T-03/05） |
| **依赖** | MC、M0 |
| **主要文件** | `app.js`（`renderTelemetryTable`）、`styles.css` |

## 目标

在**现有 Web 表格 UI**上增强列配置、列宽、格式化与右键菜单；不做桌面自绘表格、不做拖拽排序。

## Vibe Coding 任务清单

### P1 — 高优先级

- [x] **T-01** 定义列模型（`columnKey`、标题、默认显隐），表头/单元格按配置渲染
- [x] **T-01** 工具栏或入口打开「列配置」面板，勾选后即时刷新表格
- [x] **T-01** 列配置写入 `uuspace.web2.v1.telemetry.columns` 并启动加载
- [x] **T-02** 列头右缘拖拽调整宽度（`pointer` 事件 + 最小宽度）
- [x] **T-02** 列宽写入 localStorage，刷新后恢复
- [x] **T-04** 当前值列使用 MC.`formatTelemetryNumber`（含科学计数，Q-03）

### P2 — 中优先级

- [ ] **T-03** 行右键菜单：「设置小数位数」→ 自动(-1) / 0–12
- [ ] **T-03** 按行 `code` 存 `telemetry.decimals`，刷新后生效
- [ ] **T-05** 右键菜单：「列配置…」、自定义视图中「删除选中行」（若有 `tableViews`）
- [ ] **T-05** 菜单样式与现有 Web 暗色一致，不照搬桌面

### 交互修复（M1 前置）

- [x] 点击行选中不再 `renderView` 整表重绘（避免滚动条跳回顶部）
- [x] 表头 `thead` 在 `.table-scroll` 内 `position: sticky` 固定
- [x] 自定义表格重命名输入不再触发 `renderView`（防抖仅持久化 + 就地更新 tab 文案）
- [x] 勾选波道复选框不触发 `renderView`（`updateWaveSelectionInPlace`，保持表格滚动位置）

### 约束（勾选表示已遵守）

- [ ] 仍为 HTML `<table>` + 现有 Sheet / 搜索 / 筛选交互
- [ ] 仅告警行高亮（Q-02），不实现三态行色
- [ ] 未实现行拖拽、列拖拽、通道树拖入（Q-13）

### 验收

- [ ] 切换 Sheet 后列配置仍合理（按设计：全局或 per-sheet，实现时二选一并写注释）
- [ ] 隐藏「十六进制」列后刷新页面仍隐藏
- [ ] 大数/小数显示为 `e` 记数法
