## 目标

在 **Python 主后端**下，API 与 `app.js` / `realtime.js` 已用契约一致；遥测大表（Sheet 0–7）、UDP 解析、SSE、指令发送可联调。Node 栈仅作 API 对齐与 `npm test`，非遥测大表验收路径（见 `docs/项目prompt.md` §4.0）。
